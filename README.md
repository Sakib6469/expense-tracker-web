# Expense Tracker (Web)

A portfolio project demonstrating frontend development using HTML, CSS, and JavaScript, with dynamic UI updates and local storage persistence.

---

## Overview
This is a simple web-based expense tracker that allows users to add, view, and delete transactions while maintaining a real-time balance.

## Features
- Add income and expense transactions
- Real-time balance calculation
- Color-coded transactions (income/expense)
- Delete transactions
- Data persistence using localStorage

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run
Simply open `index.html` in your browser.
